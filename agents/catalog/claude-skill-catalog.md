Repository-level catalog of shared skills intended for parity.

## Global Plugin Note

Some capabilities may exist only in user-level plugin installs (for example caveman mode).
Those are global runtime capabilities, not repository-mirrored skills, unless explicitly copied into `agents/workspace/skills/`.

## Update Policy

1. Add or update skill content under `agents/workspace/skills/` first.
2. Mirror to tool-specific folders only when needed.
3. Keep this catalog synchronized with actual shared skill folders.
