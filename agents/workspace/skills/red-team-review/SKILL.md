# red-team-review

Run a focused security pass on completed or near-complete changes.

## Checklist

1. Prompt-injection and instruction-override paths.
2. Tool misuse and excessive permissions.
3. Auth and authorization gaps.
4. Unsafe file and network operations.
5. Secret exposure and logging leaks.

## Output

1. Findings by severity.
2. Recommended mitigations.
3. Residual risk summary.
